# Discord pre-sales FAQ

**Use this in Discord:** Pin this (or a link to it) in your FAQ/support channel so members see: **What plan for X?**, **How do I migrate?**, **Where’s my server?** Copy-paste the sections below into a single message and pin that, or link to this doc.

---

## What plan do I need for X?

- **Minecraft (vanilla, small):** 2–4GB is usually enough for a few friends. 4–8GB for modded or more players.
- **Minecraft (modded / plugins):** 4GB minimum; 8GB recommended for larger modpacks or 16+ players.
- **Rust:** 4GB for small servers; 8GB+ for 50+ players or heavy mods.
- **Palworld:** 4–6 players typically; 4GB or 8GB depending on player count and mods.
- **Other games:** Check our [Deploy](https://givrwrld.com/deploy) page — each game has plan options. When in doubt, start with the lowest tier and upgrade from your dashboard if needed.

---

## How do I migrate my server to GIVRwrld?

1. **Back up your current server** (world/save files, configs, mods) via your current host’s panel or FTP.
2. **Rent a GIVRwrld server** for the same game and plan size (or larger). Complete setup in the configurator.
3. **Upload your files** to the new server using the control panel (Pterodactyl) file manager or SFTP once the server is provisioned.
4. **Restore configs** (server.properties, etc.) and restart. If you use a custom egg or modpack, check our docs or ask in support for the closest match.

We don’t offer automated migration from other hosts; it’s a manual copy of your data to your new GIVRwrld server.

## Where’s my server? / I paid but don’t see my server

- **Right after payment:** Provisioning usually finishes in a few minutes. Check your **dashboard** → **My Services** (or **Order Services**). Status will go from “provisioning” to “active” when it’s ready.
- **Still not there after ~10 minutes?** Check your email for any errors, then open a **support ticket** or ask in the support channel with your order ID or email. We’ll look it up.
- **Panel access:** Once the server is active, use **“Open Panel”** on the server card in your dashboard, or go to the panel URL we sent you. Log in with your GIVRwrld account email (panel account is created at signup).
