# Canonical marketing copy

Use these lines everywhere (ads, landing pages, comparison content, affiliate pitches) so the story is consistent and we don’t overpromise.

---

## Typical time to server ready

- **Short:** “Typical time to server ready: see our [Status](https://givrwrld.com/status) page for current median.”
- **Alternative:** “Servers typically ready in a few minutes. We publish live provisioning stats on our Status page.”

*Do not* use a fixed “X minutes” unless it matches the current median on the Status page. Point to Status for the number.

---

## DDoS

- **Canonical:** “DDoS mitigation is included for all game servers.”

*Do not* use “advanced” or “enterprise” unless it’s accurate. One line is enough.

---

## 48-hour guarantee

- **Canonical:** “Not happy in the first 48 hours? Contact support for a full refund.”

Use this on checkout, FAQ, and any “why us” or comparison copy. Keep it consistent with Terms and Support.

---

## Where these appear today

- **Status page:** Typical time (from API), DDoS line.
- **Deploy, About, DeployMinecraft:** DDoS line.
- **Checkout, FAQ:** 48h guarantee + DDoS where relevant.
- **Transparency page:** All three; links to Status.

When adding new landing pages or ad copy, reuse these lines and link to [Transparency](https://givrwrld.com/transparency) or [Status](https://givrwrld.com/status) for proof.
