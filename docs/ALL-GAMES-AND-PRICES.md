## Source reference

- **Base plans:** `sql/scripts/seed-12-games-plans.sql`
- **Extra tiers for modded variants:** `sql/scripts/seed-extra-tiers-for-variants.sql` (Rust 4 GB, ARK 8 GB, Terraria 4 GB, etc.)
- **Minecraft variants:** `api/scripts/seed-minecraft-variant-plans.js` (base: 2→$6.99, 4→$13.99, 8→$27.99, 12→$36.99)
- **All other game variants:** `api/scripts/seed-game-variant-plans.js` (price = base tier `price_monthly` + variant `surcharge`, only where `tier.ram >= variant.minRam`)
- **Variant egg definitions:** `api/scripts/create-game-variant-eggs.js` (same list as above; clones source eggs in Panel)
